# Bassini Tecnologia

Sites Profissionais que Destacam Seu Negócio.

## Tecnologias Utilizadas

- React
- Vite
- TypeScript
- Tailwind CSS
- Supabase
- Shadcn UI
- Framer Motion

## Desenvolvimento

Para iniciar o servidor de desenvolvimento:

```bash
npm install
npm run dev
```

## Produção

Para gerar a build de produção:

```bash
npm run build
```
